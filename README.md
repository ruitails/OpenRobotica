# OpenRobotica
Code development for a line-following car that picks up boxes.

> **How to commit:**
> 
> 1. `git clone https://github.com/ruitails/rosa.git`
> 
> 2. `git checkout movements`
> 
> **After modifications:**
> 
> 3. `git add .`
> 
> 4. `git commit -m "Commit message"`
> 
> 5. `git push origin movements`
